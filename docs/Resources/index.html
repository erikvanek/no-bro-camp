Resources

List of resources that are relevant no-bro camp.

Sites

  • [http://www.smashingmagazine.com/](Smashing magazine) is a go-to place when looking for well curated content on building on web. One can find coding tutorials, design long reads or conference calendars. Worth following well.
  • CSS-Tricks is a bit more focused on frotend web development. Lots of articles and tutorial on Html, Javascript, CSS. Also good in depth guides on bigger topics e.g. one for Flexbox.
  • A list apart similar to Smashing magazine but articles released in slower pace. More long reads. Also releasing brief books under A book apart label.
  • Tympanus codrops is a good choice if you want to see a demo of some technology or just simply get inspired. You can find a lot of nicely crafted interactive demos and tutorials. Also worth subscribing to their newsletter.

People

  • Luke Wroblewski - experienced digital designer focusing a lot on mobile-first experiences. Quite active on Twitter.
  • Brad Frost is focusing more on web itself. Famous for his Atomic design framework
  • Jeremy Keith experienced web designer often writing and speaking about recent web development trends.
  • Rachel Andrew is famous especially for her deep insights in the world of CSS. Especially her content on Flexbox layout module and CSS grid are quite helpful.
  • Addy Osmani is currently working in Google in Chrome team and his main topic is performance and web standards. Often you might see him elaborating on progressive web apps.
